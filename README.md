# Проект "Игры разума". В данном проектре реализовано приложение, которое позволяет прокачать свой мозг некоторыми простыми математическими играми. Реализованы следующие игры: 
# 1. Приветствие. Знакомство с игрой. 
# 2. Игра "Проверка на четн"ость". В этой игре требуется отгадать, является ли число четным или нет. Выбор цифр происходит аввтоматически программой.
# 3. Игра "Калькулятор". В этой игре требуется дать ответ на простое математическое задание. Реализованы операции сложения, вычитания и уможения. Операции, как и числа выбираются случайным образом.
# 4. Игра "Наибольший общий делитель". В этой игре требуется найти наибольший общий делитель двух случайно выпавших чисел. 
# 5. Игра "Арифеметическая прогрессия". В этой игре требуется найти отсутствующее число в арифметической прогрессии. Проегрессия, как и само число, случайные. 
# 6. Игра "Простое ли число?". В этой игре требуется указать, является ли число простым или нет. Число выбирается приложением случайно.
# В каждой игре предусматриваются три раунда. Если все три раунда выиграны, то игра выиграна. Если хотя бы в одном из раундов ответили неправильно, то игра автоматически заканчивается.
